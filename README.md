# DGame

# Build
**To build this project CMake version 3.5 is required!**

**Windows:**
- create build folder
- inside build folder run cmake command, i.e. 
> cmake -G "Visual Studio 15 2017 Win64" ..
- this command will generate Visual Studio solution file for x64 arch
